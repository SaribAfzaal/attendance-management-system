# Attendance Management System

A console-based C++ application that tracks student attendance,
calculates percentages, determines exam eligibility, and computes fines.

## Features
- Accepts total classes held as user input
- Calculates attendance percentage for each student
- Determines exam eligibility (75% threshold)
- Classifies attendance status: Excellent / Good / Average / Poor
- Calculates fine for missed classes (Rs. 200 per class)
- Displays formatted individual reports
- Shows complete class summary (average, highest, lowest attendance)

## How to Run
1. Clone the repository
2. Open in any C++ compiler (VS Code with g++, Dev C++, Code::Blocks, etc.)
3. Compile and run
4. Enter total classes held when prompted
